KAPSARC Toolkit for Behavioral Analysis

A Framework for Modeling Collective Decision-Making Processes (CDMPs)

The initial model that has been developed within the KTAB framework with an accompanying user interface (UI) is the Spatial Model of Politics (SMP). Other models are available in the source code.

The KTAB SMP is available for download for Windows, Linux, and Mac OS. The code is compiled, linked, and tested with absolutely no modification, simultaneously for all OSs: there is one single body of cross-platform code, not separate versions for each OS. It was developed and tested on multiple 64-bit platforms including:

KTAB has been developed as modular libraries of code to enable analysts and computer programmers to construct their own models of CDMPs, secure in the knowledge that their approach is theoretically sound. These are already available for free download at https://github.com/KAPSARC/KTAB

Installing KTAB SMP on Windows

Installation of the SMP application on Windows is simple.
1. Download and install Microsoft Visual C++ Redistributable for Visual Studio 2015 (or higher) from here
2. Click the "Download SMP Release (Windows 64-bit)" link above to download an archive of the application compiled for 64-bit machines, along with all the required non-Microsoft shared libraries.
3. Extract the contents of the archive to your PC, maintaining the directory structure in the archive
Nothing else should be required to run the application executable KTAB_SMP.exe.

Installing KTAB SMP on Mac OS X

Installation of the SMP application on Mac OS X is simple
1. Click the "Download SMP Release (Mac OS X)" link above to download an archive of the application compiled, along with all the required shared libraries.
2. Extract the contents of the archive, maintaining the directory structure in the archive

Note for the Mac OS X version: If you need to run smpc and smpcDyn from the command line you need to navigate into the KTAB_SMP.app bundle, the executables are located in KTAB_SMP.app/Contents/MacOS/
